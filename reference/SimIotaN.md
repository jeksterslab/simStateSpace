# Simulate Intercept Vectors in a Continuous-Time Vector Autoregressive Model from the Multivariate Normal Distribution

This function simulates random intercept vectors in a continuous-time
vector autoregressive model from the multivariate normal distribution.

## Usage

``` r
SimIotaN(n, iota, vcov_iota_l)
```

## Arguments

- n:

  Positive integer. Number of replications.

- iota:

  Numeric vector. Intercept (\\\boldsymbol{\iota}\\).

- vcov_iota_l:

  Numeric matrix. Cholesky factorization (`t(chol(vcov_iota))`) of the
  sampling variance-covariance matrix of \\\boldsymbol{\iota}\\.

## Value

Returns a list of random intercept vectors.
