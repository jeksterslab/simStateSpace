# Simulate Vectors from the Multivariate Normal Distribution

This function simulates random vectors from the multivariate normal
distribution.

## Usage

``` r
SimMVN(n, mu, sigma_l)
```

## Arguments

- n:

  Positive integer. Number of replications.

- mu:

  Numeric vector. Mean vector (\\\boldsymbol{\nu}\\).

- sigma_l:

  Numeric matrix. Cholesky factorization (`t(chol(sigma))`) of the
  variance-covariance matrix \\\boldsymbol{\Sigma}\\.

## Value

Returns a list of random vectors.

SimMVN(n = n, mu = mu, sigma_l = sigma_l)
